Tools

In this assessment, much like assignment 2, we have used a number of tools to develop, collaborate, and communicate on the project. Some of these tools are specific only for the completion of work, but some of these tools were also used to produce artefacts and prototypes. For the completetion of the project, there are more tools within the project details page.

  • Microsoft Teams: A collaboration environment allowing users to share thoughts and files.
    • We used MS Teams to have weekly meetings and discuss the tasks at hand
    • Shared files and feedback using the programs file sharing functionality
  • GitHub: A code hosting platform for version control and collaboration.
    • GitHub was used for version control on the artefact website, as well as Pages for the assignment submission.
  • Microsoft Word: A word processing software.
    • We used Microsoft Word to develop our reports and research.
  • Visual Studio Code: A source-code editor for use with a variety of programming languages.
    • We used Visual Studio Code to develop our websites HTML and CSS
  • Visual Studio 2022: An integrated development environment.
    • Visual Studio 2022 was used to make software prototypes.
  • Adobe XD: An application design prototyping tool.
    • Adobe XD was used to produce prototype mockups for the application
  • SQL Server Management Studio: A database management tool.
    • SSMS was used to create tables as artefacts for the project

For this assignment, since much of the focus was on developing and planning our project idea with reusable content from A2, GitHub was not utilised as much for the development of our presentation. It was however used in some of the prototyping, but for this assignment the GitHub metrics do not show any real indication of how the group shared the workload. Since the workload was assigned based on our strengths, GitHub related tasks were left with Leigh who has the most experience and familiarity with it.
